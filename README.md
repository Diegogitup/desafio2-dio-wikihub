# Wiki de Repositórios GitHub

Uma aplicação desenvolvida com React que permite pesquisar, visualizar e gerenciar repositórios do GitHub de forma simples e intuitiva.

## Funcionalidades

- Buscar repositórios através da API do GitHub.
- Exibir informações relevantes de cada repositório.
- Adicionar repositórios a uma lista personalizada.
- Remover repositórios da lista.
- Evitar duplicidade de repositórios cadastrados.
- Interface moderna e responsiva.

## Tecnologias Utilizadas

- React
- JavaScript
- Styled Components
- API do GitHub
- HTML5
- CSS3

## Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais do React, como gerenciamento de estado, consumo de APIs, componentização, eventos e manipulação dinâmica de listas.

## Demonstração

Pesquise um repositório informando o formato:

```
usuario/repositorio
```

Exemplo:

```
facebook/react
```

Após a busca, o repositório será adicionado à lista, permitindo consultar suas informações e removê-lo quando desejar.
